Скрипт для создания образа raw из физических дисков linux.
Кратко:
1. Выбираем нужный физический диск.
2. Вводим данные:
 - шара куда лить
 - куда шару монтировать
 - учетные данные с которыми подключаться к шаре
! Для подключения используется mount cifs
!! Не нужно создавать директорию для монтирования, создается сама
3. Запуск создания образа.
!!! Пока идет создание образа, ОС находится в состоянии freeze. Может работать только на чтение.
